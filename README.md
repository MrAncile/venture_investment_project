## В рамках анализа венчурных инвестиций выполнил:

- Предобработку данных: очистил датасет от дубликатов и пропусков, проверил корректность числовых и временных значений.

- Сегментацию компаний: сгруппировал по срокам финансирования, выделил массовые, средние и нишевые сегменты рынка, исключил выбросы и ограничил период исследования.

- Анализ финансирования: сравнил популярность и объёмы разных типов финансирования, изучил динамику раундов и инвестиций по годам.

- Оценку возврата средств: рассчитал ROI для разных типов финансирования и оценил устойчивость показателей.

- Формулировку рекомендаций: на основе анализа подготовил предложения по инвестициям на 2015 год — определил приоритетные сегменты рынка, типы финансирования и оптимальные объёмы вложений.
